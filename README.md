🍯 Honeydew

Honeydew is a fun and interactive Slack bot made to make your workspace a little more entertaining.

Users can interact with Honeydew and get random cat moods and quotes, intresting cat facts or challenge themselves with an animal quiz

💬 How to Use Honeydew
You can chat with Honeydew by sending it messages such as:

Hello.
Tell me a cat fact.
What is the cat mood.
Give me a cat quote. 
Qiuz me.

🛠️ Built With
JavaScript — Main programming language
Node.js — Runtime environment
Slack Bolt — Framework used to build the Slack bot
Axios — Used to make API requests
dotenv — Used to manage environment variables
GitHub — Used for version control and hosting the source code
